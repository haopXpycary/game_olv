## 帮助文档

---

##### [ 运行 ]
环境要求: 
系统:  GNU/Linux | Unix
语言:  python3 
```bash
$ cd game_olv/pyRail/
$ chmod +x main.py
$ python3 main.py
```

- pyRail&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;存放python脚本文件及外部库
